🚀 Turborepo with shadcn/ui Components
 A monorepo setup using Turborepo, Vite, Tailwind CSS v4, and shadcn/ui components.

Install dependencies:

pnpm install
Build


Start all repos at once:

pnpm run dev

Run Apps Individually

Run the admin app:

pnpm admin


Run the staff app:

pnpm staff
